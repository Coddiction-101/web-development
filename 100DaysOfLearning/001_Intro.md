## Understanding Web Development
- The **Web** is a system of interconnected documents and resources (like articles, images, videos, and MP3s) that can be accessed using the **Internet**.       
- The Internet is a larger entity, and the Web is a smaller subset or service within it.       
- **Web Development** is the process of creating websites and web applications.       

### Website vs. Web Application
- The choice between creating a website or a web application depends on the specific use case and needs.      
- This course primarily targets the creation of web applications.     

| Feature             | Website                                     | Web Application                               |
|:------------------ |:------------------------------------------ |:-------------------------------------------- |
| **Content Nature**  | Static, fixed content                       | Dynamic, content changes based on interaction |
| **User Interaction**| Minimal or negligible (read-only)           | High level of user interaction possible       |
| **Examples**        | Wikipedia pages, blogs                      | LinkedIn, Instagram, Facebook                 |
           

## Web Application Architecture
- A complete web application consists of multiple interconnected layers.    

  
                       

- Professionals specializing in each layer are called **Frontend Engineers**, **Backend Engineers**, or **Database Administrators**.    
- A **Full Stack Developer** is an engineer proficient in working across all three layers: frontend, backend, and database.     

## MERN Stack for Full Stack Development
- This course utilizes the **MERN Stack** for building web applications.   

| Component | Full Form   | Role                                     |
|:-------- |:---------- |:--------------------------------------- |
| **M**     | **MongoDB** | Database for storing application data    |
| **E**     | **Express.js** | Backend web application framework (used with Node.js) |
| **R**     | **React**   | JavaScript library for building user interfaces (frontend) |
| **N**     | **Node.js** | JavaScript runtime for building the backend |
